How to regenerate OBF files in this folder:

1. Open GPX file in JOSM (do not load any data before opening)
2. Click "Download from OSM along this track" on the layer
3. Choose "100 meters" in "Download everything within"
4. Click "Save" on the "Data" layer, save as OSM
5. Convert OSM to OBF with OsmAndMapCreator
